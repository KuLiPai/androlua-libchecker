# androlua-libchecker
An androlua project modeled after LibChecker is based on OpenLuaX+

### 项目简介

本项目受 [LibChecker](https://github.com/LibChecker/LibChecker) 启发，使用 AndroLua 开发。

### 运行环境

要运行此项目，您需要在 Android 设备上安装 [OpenLuaX+](https://github.com/znzsofficial/OpenLuaX_Open-Source)。

### 关于

本项目最初于 2023 年 2 月完成开发，但由于后续时间有限，一直未能继续完善，项目因此搁置了很长时间。最近在整理网盘时偶然发现了该项目，决定将其开源，希望能为 AndroLua 开发者的学习和探索提供一些帮助。

目前我已基本退出 AndroLua 开发圈，但仍会偶尔使用 AndroLua 完成一些小型项目。欢迎大家在此基础上进行改进或提供建议！
